# Kako kreirati zadatak za došašće++?

Ako imaš ideju za zadatak i želiš doprinijeti natjecanju, slijedi ove upute:

1. Osmisli zadatak: Zadatak mora imati ogroman problem scope (>> 10<sup>10</sup>). Postoji teoretski optimalno rješenje, ali ga je gotovo nemoguće pronaći u realnom vremenu. Cilj je osmisliti algoritam koji pronalazi što bolje rješenje u ograničenom vremenu.
2. Pripremi podatke: Trebat će nam opis zadatka (.md), jednostavni primjer ulaza (.txt) i izlaza (.txt) i checker (Python) koji provjerava točnost rješenja. Također pripremi neku naslovnu sliku za zadatak.
3. Javi nam se putem emaila: <Anchor href="mailto:dosasce@xfer.hr">dosasce@xfer.hr</Anchor>.

Detaljnije upute za formatiranje i slanje zadataka bit će objavljene uskoro. Hvala na interesu!
